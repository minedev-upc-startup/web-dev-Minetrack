# webdev-course-Minetrack

# Fundamentos de Desarrollo Web

## Resumen del Curso
Este proyecto forma parte del trabajo complementario del curso **Aplicaciones Web** de la **Universidad Peruana de Ciencias Aplicadas (UPC)**.  
El objetivo es diseñar un curso en video de aproximadamente 1 hora para enseñar los fundamentos del desarrollo web (HTML y CSS) a estudiantes de secundaria, usando como proyecto final la creación de un **Blog de Animes**.

El curso se compone de videos cortos, prácticas interactivas en el editor online **CodePen** y un repositorio con ejemplos de código.

**Público objetivo:** Estudiantes de 12 a 17 años sin experiencia en programación
